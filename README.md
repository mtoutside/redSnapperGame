# Red Snapper Game

Bite! Bite! Bite! Bite small fishes! Avoid enemy!

![](header.png)

## How to play

Manipulate red snapper and eat small fishes. However, be careful with green fish...

 - UP ARROW: boost
 - LEFT ARROW: turn the left
 - RIGHT ARROW: turn the right
